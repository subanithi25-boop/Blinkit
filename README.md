📊 Blinkit Sales Analysis Dashboard (Power BI)
🚀 Project Overview

This project is a Power BI dashboard built to analyze the sales performance of Blinkit (India’s last-minute delivery app).
It provides insights into total sales, average sales, item types, outlet performance, and customer ratings to help understand business growth and decision-making patterns.

📌 Key Features
📈 Total Sales Analysis – Overall revenue performance ($1.20M)
🧾 Average Sales Tracking – Average sales value per transaction ($141)
🛒 Item Insights – Analysis of ~9K items across categories
⭐ Customer Rating Analysis – Average rating visualization (4.0)
🏬 Outlet Performance – Sales comparison across outlet types & sizes
📍 Location-Based Insights – Sales distribution by Tier 1, Tier 2, Tier 3 cities
🍔 Fat Content Analysis – Low Fat vs Regular product performance
📊 Year-wise Trends – Sales trends by outlet establishment year
📊 Dashboard Visuals

The dashboard includes:

KPI Cards (Total Sales, Avg Sales, No of Items, Ratings)
Line Chart (Outlet Establishment Year vs Sales)
Donut Charts (Fat Content distribution)
Bar Charts (Item Type & Outlet Location analysis)
Matrix Table (Outlet Type performance comparison)
🧠 Business Insights
Tier 3 locations contribute the highest sales share.
Supermarket Type 1 performs the best among outlet types.
Fruits, snacks, and household items dominate sales.
Majority of sales come from low-fat products.
Sales peaked around specific outlet establishment years.
